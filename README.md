 🧠 AI Pose Estimation System

 Project Overview

This project is a real-time AI-based pose estimation system developed using web technologies. It detects human body keypoints from live webcam video and visualizes them as a skeleton structure. The system also includes posture detection and a squat counter for fitness tracking.

---

 Features

* Real-time pose detection
* Skeleton visualization
* Posture analysis (Correct / Wrong)
* Squat counter
* Animated and responsive user interface

---
  Technologies Used

* HTML, CSS, JavaScript
* TensorFlow.js
* MoveNet Model
* Git & GitHub

---

 How to Run the Project

1. Download or clone the repository

2. Open terminal in project folder

3. Run the following command:

   npx serve

4. Open browser and go to:

   http://localhost:3000

5. Allow camera access 

---

 Output

* Displays live camera feed
* Shows body keypoints as red dots
* Connects joints with skeleton lines
* Displays posture status
* Counts squats in real-time
